### 特点：
1、不回头
2、贪婪模式（能匹配多绝不匹配少）


#### 量词：
n+ ：[1, 正无穷]
n* ：[0, 正无穷]
n? : [0, 1]
<!-- 注意，y 前面不能有空格！！！ -->
n{x,y} ：[x, y]
<!-- 上面特例： -->
n{x,} ： [x, 正无穷]
因此：n+ === n{1,} n* === n{0,}
^n ：匹配任何以 n 开头的字符串
n$ ：匹配任何以 n 结尾的字符串
?=n ：匹配任何其后紧接着指定字符串 n 的字符串
?!n ：匹配任何其后紧接着不是指定字符串 n 的字符串




例题：
```js
    var reg = /\d*/g
    let str = 'jfion'
    console.log(str.match(reg));
```
打印结果是什么呢？
['', '', '', '', '', '']


```js
    var reg = /\w?/g
    let str = 'jfion'
    console.log(str.match(reg));
    //  ['j', 'f', 'i', 'o', 'n', '']
```
